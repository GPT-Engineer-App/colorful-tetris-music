# colorful-tetris-music

We want to have a browser based Tetris game, which can be controlled via the arrows key. Every time we make line, the background color of the game should change.
The tetris game should follow the normal tetris rules. Please add tetris original music from the gameboy.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/colorful-tetris-music.git
cd colorful-tetris-music
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
